# Math-ML-Project
I used Micael Nielson's 2015 book as an aid to make a network to recognize handwritten digits trained from the MNIST database. I also made a 3d plot to illustrate gradient descent.
This code was developed as a supplementary practical application to verify the five fundamental equations of backpropagation outlined in the main LaTeX submission.
The neural network is built entirely relying on NumPy for matrix operations and linear algebra. No high-level machine learning frameworks (like PyTorch or TensorFlow) were used.
The model achieves ~96% classification accuracy on the MNIST handwritten digit test set.
Includes the script used to programmatically generate the 3D surface plots of the gradient descent cost function featured in the paper.
